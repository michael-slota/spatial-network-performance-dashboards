5G Coverage Dashboard Suite

This website presents interactive analysis of 5G network measurements collected across a city around a central tower cluster. The dashboards explore how signal quality, throughput, latency and packet loss vary with distance and direction, and compare the measurements with target and minimum acceptable service levels.

1D Analysis

Distance Performance Dashboard

Explore how each 5G performance metric changes with distance from the tower cluster. This dashboard includes fitted distance models, service-level lines, coverage-radius summaries and controls for switching between signal, throughput, latency and packet-loss metrics.

Launch 1D Dashboard
2D Analysis

Directional Coverage Dashboard

Inspect citywide coverage by direction and distance ring using polar plots. The dashboard reports the percentage of measurements meeting the target service level or the minimum acceptable service level for each metric and frequency band.

Supporting Visualisations

These additional plots show the underlying tower-cluster and citywide measurement data used to build the main dashboards. They are useful for checking measurement locations, comparing raw metric values and inspecting distance patterns before opening the larger dashboard views.
